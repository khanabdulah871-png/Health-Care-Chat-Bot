# HealthCare Chat Bot

##Project Overview

HealthCare Chat Bot is an AI-powered application that provides general information about medical conditions, treatments, and healthy lifestyle practices. The chatbot is built using the Groq API and deployed with Streamlit for an interactive user interface. This project is designed for educational purposes and does not replace professional medical advice.


##Features

* AI-based healthcare assistance
* Provides general medical information
* Suggests lifestyle and wellness tips
* Interactive web interface using Streamlit
* Secure API key management using environment variables

##Technologies Used

* Python
* Streamlit
* Groq API
* dotenv (for environment variables)
